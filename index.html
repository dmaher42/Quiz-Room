🔒 Private Room System

Perfect for Multiple Teachers!

  • 🎲 Unique Room IDs: Each teacher gets their own private room (e.g., ABC123, XYZ789)
  • 🔒 Complete Isolation: No interference between different classes
  • 📋 Easy Sharing: Copy your Room ID and share with students
  • 🔄 Generate New: Create fresh rooms anytime for different classes

🚀 Advanced Game Modes Added!

  • 💰 Wager System: Teams bet points based on confidence level
  • 🏆 Bonus Rounds: Special high-value questions worth double points
  • 🤝 Team Challenges: Collaborative puzzle rounds with progress tracking
  • ⚡ Quick Fire Mode: Rapid-fire questions with countdown timers
  • ⏱️ Timer System: Set 30s, 60s, or 90s countdowns for questions
  • 📝 Multiple Choice: Create A/B/C/D questions with visual options

🎮 How to Use Game Modes:

  1. Standard Mode: Regular buzzer questions with timers
  2. Wager Mode: Teams bet points before answering
  3. Bonus Round: High-stakes questions worth double points
  4. Team Challenge: Collaborative questions where teams work together
  5. Quick Fire: Fast-paced rounds with 10-second timers

  • 🛡️ Password Protection: Presenter Panel and Quiz Builder are now locked
  • 🔑 Default Password: "TEACHER" (change this in Security Settings above)
  • 👨‍🎓 Student Access: Only Player Buzzer mode is available without password
  • 🔄 Per-Room Passwords: Each room can have its own unique password
  • ⚡ No Cheating: Students cannot access quiz answers or controls

🚀 How to Deploy This Quiz

  1. Click "Use in a design" above - This opens Canva editor
  2. Click "Publish website" - Get your live URL instantly
  3. Share URL with colleagues - Each gets their own private room
  4. Teachers use password "TEACHER" - Access Presenter Panel (change password in settings)
  5. Students use Player Buzzer - No password needed, join with Room ID
  6. All devices sync automatically - Real-time buzzing within each room!

🔐 Security Best Practices

  1. Change Default Password: Use Security Settings above to set a unique password
  2. Share Password Securely: Only give the password to teachers, not students
  3. Different Passwords: Each room can have its own password for extra security
  4. Student Instructions: Tell students to only use "Player Buzzer" mode
